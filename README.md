# MobileNetV3-Light-Semantic-Segmentation-VOC-2012-
Lightweight semantic segmentation on PASCAL VOC 2012 using MobileNetV3-Small. Custom encoder-decoder with attention, depthwise separable convolutions, ~1.2M params, &lt;0.5 GFLOPs. Trained with CrossEntropy + Dice loss. Supports CUDA, CPU &amp; Intel DirectML.
